# 🛍️ Tech-Aesthetics Inventory & Sales Platform

## 📌 Project Overview

The **Tech-Aesthetics Inventory & Sales Platform** is a full-stack MERN web application developed to transform a Facebook-based e-commerce business into a structured, automated digital system.

The platform manages the complete lifecycle of imported gadgets and minimalist decor products — from Chinese suppliers to Bangladeshi customers — using a structured MongoDB database.

This project strictly follows the **MVC (Model–View–Controller)** architecture to ensure maintainability, scalability, and academic-standard software quality.

---

## 🏗️ Architecture (MVC)

- **Model** → MongoDB schemas, inventory logic, cost tracking  
- **View** → React frontend (product gallery, cart, dashboards)  
- **Controller** → Express.js business logic & API handling  

---

## 🚀 Core Features

### 📦 Advanced Inventory & Pre-order Management
- Product onboarding with Chinese import cost & Bangladeshi retail price
- Real-time stock tracking with automatic updates
- Pre-order toggle for out-of-stock products
- Estimated arrival date display
- Low-stock alert system

### 🛒 Customer Experience & Smart Cart
- Responsive aesthetic product gallery
- Pre-order booking checkout flow
- Smart cart validation (prevents overselling)
- Automatic order summary generator
- Live order status tracking (Importing → Delivered)

### 🚚 Delivery & Logistics Management
- Admin active order queue
- Courier tracking ID integration
- Order status history audit trail
- Order cancellation logic
- Multiple address management system

### 📊 Business Analytics & Admin Control
- Revenue dashboard (Daily & Monthly performance)
- Profit margin calculator (Retail - Import cost)
- Premium membership / loyalty tracking
- Advanced product search & filtering
- Export sales data (CSV / PDF)

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js + Express.js  
- **Database:** MongoDB  
- **Architecture Pattern:** MVC  
- **Charts & Visualization:** Chart.js
